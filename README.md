# TaskPilot

TaskPilot is a responsive project task manager built for the WDD 430 final project using Next.js, React, and TypeScript.

## Features

- Create, edit, and delete tasks
- Move tasks through To do, In progress, and Completed statuses
- Assign priorities and due dates
- Search tasks by title or description
- Filter tasks by status and priority
- View live project statistics and completion progress
- Automatically save tasks in the browser
- Export all task data as JSON
- Responsive desktop and mobile design


## Deploy to Vercel

1. Push this project to a public GitHub repository.
2. Sign in to Vercel and select **Add New → Project**.
3. Import the GitHub repository.
4. Keep the detected Next.js settings and select **Deploy**.

## Technology

- Next.js App Router
- React
- TypeScript
- CSS
- Browser local storage
